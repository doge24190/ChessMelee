开始界面→选择难度（9×9/14×14/19×19棋盘）→进入对局。

核心玩法：

* 基本规则：黑棋先行，双方轮流在棋盘上执行落新子或移动己方棋子两种操作（每回合100秒倒计时）；棋子包含普通子与马、炮、兵等特殊子（特殊子具备直线、斜线、跳跃等移动能力且数量有限）。
* 连击系统：落子后若横竖斜方向形成2连触发"夹击"吃掉相邻敌子，3连奖励随机特殊棋子，4连进入选择状态（吃敌子或额外落一子），5连整条连珠变为"灵体"得分棋子。
* 胜负判定：当棋篓空置或棋盘填满时结束，拥有更多灵体棋子的一方获胜，否则比较普通棋子数量。

单人模式提供三级AI难度（简单随机/中等评估/困难预判），支持完整UI菜单与音效控制。
